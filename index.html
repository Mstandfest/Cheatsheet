HTML/CSS Cheatsheet

HTML Cheatsheet

Textformatierung

Name Bezeichnung Code Erklärung
Textspanne span <span></span> Formatieren einzelner Textteile
Fett strong <strong></strong> Fett gedruckter Text
Kursiv italic <i></i> Kursiv gedruckter Text
Betont emphasis <em></em> Betonter Text
Link anchor <a></a> Einfügen von URLs und interner Links
Absatz break <br/> Zeilenumbruch im Text, Achtung: kein End-Tag

Tabellen

Name Bezeichnung Code Erklärung
Tabellenkopf table head <thead></thead> Kopfzeile der Tabelle
Tabelle table <table></table> Benötigter Starttag Tabelle
Tabellenzeile table row <tr></tr> Neue Reihe in der Tabelle
Tabellenkörper table body <tbody></tbody> Dargestellter Bereich der Tabelle
Tabellenelement table data <td></td> Datenzelle innerhalb der Tabelle

Struktur

Name Bezeichnung Code Erklärung
Navigation nav <nav></nav> Navigationselement oben auf der Seite dargestellt
Seitenkopf header <header></header> Titel und Links zu Fonts oder CSS hier einfügen
Hauptteil main <main></main> Zur Unterteilung des Hauptinhaltes der Seite
Abschnitt article <article></article> für SEO abgegrenzter Teil mit Infos, Bildern oder Videos
Abschnitt (für Text) section <section></section> für SEO abgegrenzer Text
