CSS Cheatsheet

Text

Name Property Werte Erklärung
Textausrichtung text-align left, right, center, justify Richtet den Text entsprechend der Einstellungen aus
Schriftart font-family Liste an Schriftarten mit Komma getrennt Setzt bestimmte Schriftarten für den entsprechenden Abschnitt
Schriftstärke font-weight normal, bold, lighter, bolder Stärke der Buchstaben
Schriftgöße font-size bevorzugt x em oder x px Setzt Schriftgröße für entsprechenden Abschnitt
Textverzierung text-decoration-style dotted, dashed, solid, double, wavy Modifiziert Darstellung von Text
Textfarbe color hsl(), rgb(a)(), hex oder englische Farben Ändert die Farbe des Textes

Abstände, Positionierung und Größen

Name Property Werte Erklärung
Abstand nach außen margin px oder % top, bottom, left, right
Abstand nach Innen padding px oder % top, bottom, left, right
Position der Elemente position static, relative, fixed, absolute, sticky top, bottom, left, right (nicht bei static)
Ausrichtung float none, left, right Ausrichtung von Text
Höhe/Breite height/width px oder % Höhe von Text oder Element

Ränder

Name Property Werte Erklärung
Randstil border-style dotted, dashed, solid, double, groove, ridge, inset, outset, none, hidden benötigt für weitere Border-Settings
Randdicke border-width bevorzugt px Stärke des Rahmens
Randfarbe border-color hsl(), rgb(a), hex oder englische Farben Farbe des Randes
Randrundung border-radius bevorzugt px abgerundete Ecken
einzelne Ränder border-top/bottom/left/right-x siehe Oben Einzelne Ränder, sowohl style, als auch border-radius
